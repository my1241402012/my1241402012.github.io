个人博客

前提环境

  1. git
  2. node
  3. github

git配置以及生成ssh秘钥

  • 查看你的配置
  • git config –list
  • 在全局配置 name 和 email
  • git config -global user.name “xxx”
  • git config -global user.email “xxx@xxx”
  • 生成秘钥
  • 在命令行执行 ssh-keygen -t rsa -C “上面配置的邮箱” 回车即可
  • 注：在此路径 –> C/user（用户）/联想/.ssh/id_rsa.pub ，打开 id_rsa.pub 文件，复制文件所有内容
  • 打开github
  • settings > SSH and GPG -> new SSH key
  • title 文本框空置
  • key 粘贴刚才复制的东西

搭建博客

  1. 创建一个远程仓库，名称固定 “你的用户名.github.io”
  • 点击SSH 复制连接
  1. 本地命令行
    • 执行 npm i -g hexo-cli hexo
    • 本地创建一个文件夹
    • cd 进入文件夹
    • 打开 git bash
      • 执行 hexo init
      • 执行 npm install
      • 执行 hexo server 或者 hexo s
    • 可以在本地开启一个服务
    • 访问 localhost:4000
    • 看到本地个人博客页面
    • 执行 hexo g
    • 执行 hexo d
    • 如果执行 hexo d 的时候报错
    • 就执行 npm i hexo-deployer-git
    • 然后重复 hexo g 和 hexo d
    • 这个时候远程仓库刷新就有文件了
    • 远程仓库 点击 setting
    • 向下翻 会有一个网址
    • 网址就是 https://你的用户名.github.io/
  2. 使用hexo
    • 在本地文件夹中打开 git bash
      • 执行 hexo new 文件名
      • 在根目录下/source/posts/文件名.md
      • 编辑文件内容
      • 编辑完成以后
        • 执行 hexo g
        • 执行 hexo d
      • 网址刷新就有了（注意缓存）
